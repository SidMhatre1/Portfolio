F1 Somethings

This was a group project, a web application using Flask, HTML, Python, and JavaScript, it enables users to access Formula 1 info through customized searches. For example, users can search for Drivers by name via the Search by Driver page and get basic info about those drivers. As the project manager, my responsibilities included:

  • Coding features: Search by Driver, Search by Driver Standings, Search by Constructor, and Search by Constructor Standings.
  • Organizing and leading group meetings.
  • Setting deadlines and conducting code reviews.
  • Presenting the project.
  • Producing a comprehensive demo video.
  • Troubleshooting implementation issues.
  • Collaborating with team members on deployment via PythonAnywhere.

We used the Ergast f1 API for this project unfortunately in Dec 2024 it will be shut down but my demo video shows every feature in action and the project GitHub has all of our code!

Tacops

NETSOC (Networking & IT Security Student Association) run CTF Event (2023,2024)

Tacops is a Capture the Flag (CTF) event, blending networking challenges, trivia, and team collaboration. In a CTF, participants, like teams such as "The SS. Sal" compete to solve a series of cybersecurity puzzles or challenges, symbolized as "flags." These challenges test various technical skills, including networking, cryptography, and system exploitation. The goal is to capture as many flags as possible within the given timeframe, demonstrating proficiency in cybersecurity concepts and problem-solving at the end the team with the most flags or points wins. Tacops offered a platform for participants like myself to hone their skills, work together, and enjoy the thrill of competition.
